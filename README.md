koipy是一个Telegram 节点测速、连通性测试机器人，同时提供对接miaospeed后端的开源实现。 koipy是fulltclash的下游分支。

## 功能特性

- 多配置文件参数的配置
    * license
    * bot-token
    * api-id
    * api-hash
    * slaveConfig
    * ***etc.***

## 系统要求

经过测试支持**Docker version 20.10.23+**，其他版本未测试，理论支持所有docker版本

## 架构支持

x86_64 / AMD64，不滋瓷ARM架构
